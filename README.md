### Локальный запуск 

Склонить проект:

```bash
  git clone https://github.com/ClaqP/yandex_task.git
```

Войти в рабочую директорию:

```bash
  cd yandex_task
```

Запустить тесты:

```bash
  pytest -s -v --browser_name=chrome .\tests\test_yandex_disk_page.py
```
